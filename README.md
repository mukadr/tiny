# tiny

My first compiler including a recursive descent parser, type checking and code generation for a little VM.

Written in January 2007.

## Prerequisites

* GCC or Clang
* GNU Make

## Compiling Tiny

```
cd Tiny
make
```

## Compiling TM

```
cd TM
make
```

## Running an example

```
tiny fibonnaci.tny fibonnaci.tc
tm fibonnaci.tc
```
